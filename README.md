# Análise de Dados do Setor Financeiro com Power BI

## Sobre o Projeto

Este projeto foi desenvolvido utilizando **Power BI** com foco na análise de indicadores financeiros empresariais.  
O dashboard apresenta uma visão estratégica das principais métricas financeiras, permitindo acompanhar receitas, despesas, margem de lucro e desempenho por segmento.

O objetivo principal do projeto é transformar dados financeiros em informações visuais claras e úteis para tomada de decisão.

---

## Tecnologias Utilizadas

- Power BI
- DAX
- Excel
- Modelagem de Dados
- Visualização de Dados

---

## Principais Indicadores

O dashboard apresenta os seguintes KPIs:

- Total de Receita
- Total de Despesas
- Margem de Lucro
- Receita por Componente
- Despesas por Categoria
- Segmentação Financeira
- Comparativo por Ano

---

## Funcionalidades do Dashboard

### Indicadores Financeiros
Exibição dos principais indicadores financeiros da empresa de forma rápida e visual.

### Análise por Segmentos
Utilização de segmentação para identificar padrões financeiros e distribuição dos dados.

### Receita por Componente
Análise visual das principais fontes de receita da empresa.

### Despesas por Categoria
Visualização das despesas organizadas por componente financeiro.

### Comparativo Temporal
Tabela comparativa entre diferentes anos para análise da evolução financeira.

## Insights Obtidos

Durante a análise foi possível identificar:

- A principal fonte de receita da empresa está concentrada em vendas.
- Despesas administrativas representam uma parcela significativa dos custos.
- A margem de lucro apresentou bom desempenho em relação às despesas totais.
- Alguns segmentos possuem maior potencial financeiro do que outros.
